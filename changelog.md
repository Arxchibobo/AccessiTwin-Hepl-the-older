# Application Changelog / 更新日志

## v1.3.0 - Optimization & Sound / 优化与音效
- **Default Fast Mode / 默认快速模式**: App now launches directly into Fast Mode for immediate safety. (应用现在直接进入快速模式，确保即时安全)
- **Smart Intro / 智能介绍**: Intro speech now explains the 10 key capabilities. (开场白增加十大功能场景介绍)
- **Voice Interruption / 语音打断**: Speaking during the intro immediately stops the intro. (在介绍时说话可立即打断介绍)
- **Sound Effects / 音效反馈**: Added feedback beeps for buttons in Video Mode. (视频模式增加按键音效)
- **Bilingual Docs / 双语文档**: Features and logs are now in EN/CN. (功能与日志文档支持中英双语)
